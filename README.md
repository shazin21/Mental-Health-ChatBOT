# 🧠 Mental Health Chatbot  

A conversational AI chatbot designed to offer **mental health support**, answer common mental health-related questions, and engage users in meaningful conversations. Built using **Natural Language Processing (NLP) and Deep Learning**.  

---

## 🚀 Features  
✅ AI-powered **mental health support**  
✅ **NLP-based intent recognition**  
✅ **Predefined responses** for anxiety, depression, and stress  
✅ **Flask-based API** for deployment  
✅ **Deep Learning model** using **TensorFlow/Keras**  

---

## 📂 Project Structure  
```
├── app.py                # Flask web app (backend)  
├── training.py           # Model training script  
├── training.ipynb        # Jupyter Notebook for training (optional)  
├── intents.json          # Dataset containing predefined intents & responses  
├── model.h5              # Trained deep learning model  
├── texts.pkl             # Tokenized words data  
├── labels.pkl            # Labels for classification  
├── static/               # Folder for static assets (if any)  
├── templates/            # HTML templates for the chatbot UI  
└── README.md             # Project documentation  
```  

---

## 🛠 Tech Stack  
- **Python** (Flask, NLTK, NumPy)  
- **Machine Learning** (TensorFlow/Keras)  
- **Natural Language Processing (NLP)**  
- **Data Preprocessing & Tokenization**  

---

## 🎯 How It Works  
1️⃣ **User sends a message**  
2️⃣ **NLP model predicts intent**  
3️⃣ **Chatbot selects the best response** based on intent  
4️⃣ **Response is sent back to the user**

## 🚀 Installation & Setup  

### 🔹 **1. Clone the Repository**  
```bash
git clone https://github.com/yourusername/mental-health-chatbot.git
cd mental-health-chatbot
```  

### 🔹 **2. Install Dependencies**  
```bash
pip install -r requirements.txt
```  

### 🔹 **3. Train the Model**  
```bash
python training.py
```  

### 🔹 **4. Run the Chatbot**  
```bash
python app.py
```  
Visit `http://127.0.0.1:5000/` in your browser.
